♟️ Chess-Roulette

Your chess application with video chat is ready to play!

✅ Build Status: Successfully compiled and ready to run!

1Prerequisites

Make sure you have Node.js installed (version 18 or higher).

Check by running: node --version

If not installed, download from: nodejs.org

2Open Terminal

Navigate to the chess-roulette folder:

cd chess-roulette

3Install Dependencies

Run this command (only needed once):

npm install

This downloads all required packages (~165 packages).

4Start the App

Run the development server:

npm run dev

You'll see output like:

VITE v6.4.3 ready in 500 ms ➜ Local: http://localhost:5173/ ➜ Network: use --host to expose

5Open in Browser

Open your browser and go to:

http://localhost:5173

Tip: Open this URL in two browser windows to test multiplayer!

6Play Chess!

  • Click "Play" to start matchmaking
  • Wait for another player to join
  • Enable camera/microphone when prompted (optional)
  • Play chess with video chat!
⚠️ Video Chat Note: Chess gameplay works perfectly! Video chat may require TURN server credentials for some networks. To enable video for all users, get Cloudflare Realtime TURN credentials and update wrangler.jsonc.

🚀 Optional: Deploy to Cloudflare

To get a live URL you can share with anyone:

npm run deploy

This will give you a URL like:

https://chess-roulette.your-subdomain.workers.dev
